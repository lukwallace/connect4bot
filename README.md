# connect4bot
> a small bot for slack!

#Dependencies
- Node & npm

#Usage
Make a bot integration in side your Slack channel and [get a Bot API token from the Slack integrations page](https://my.slack.com/services/new/bot).

Fork this repo.

Inside the root folder of your fork:
```sh
npm install
token=[YOUR_BOT_API_TOKEN_HERE] node connect4bot.js
```
#Instructions for play
Currently only supports one game.

In a direct message to your bot:
  play @name_of_opponent
